バリデータリファレンス¶

Note

  • 対象バージョン：2.3
  • 翻訳更新日：2013/6/7

バリデータは、制約 に対してオブジェクトを検証するように設計されています。 現実の世界では、制約はたとえば「ケーキは焼いてはいけない」となります。 Symfony2 の世界でも同様です。 制約とは、特定の状況を満たしていることを表明するものです。

サポートしている制約の一覧¶

Symfony2 では、次の制約が組み込みで利用できます。

基本制約¶

以下は基本的な制約で、オブジェクトのプロパティの値やメソッドの戻り値に関する基本的なことを検証する場合に使います。

  • NotBlank
  • Blank
  • NotNull
  • Null
  • True
  • False
  • Type

文字列制約¶

  • Email
  • Length
  • Url
  • Regex
  • Ip

数字制約¶

  • Range

比較制約¶

  • EqualTo
  • NotEqualTo
  • IdenticalTo
  • NotIdenticalTo
  • LessThan
  • LessThanOrEqual
  • GreaterThan
  • GreaterThanOrEqual

日付制約¶

  • Date
  • DateTime
  • Time

コレクション制約¶

  • Choice
  • Collection
  • Count
  • UniqueEntity
  • Language
  • Locale
  • Country

ファイル制約¶

  • File
  • Image

会計およびその他の番号制約¶

  • CardScheme
  • Luhn
  • Iban
  • Isbn
  • Issn

その他の制約¶

  • Callback
  • Expression
  • All
  • UserPassword
  • Valid
