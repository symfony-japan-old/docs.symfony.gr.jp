Form Type リファレンス¶

フォームは フィールド で構成されています。 各フィールドは、text タイプや choice タイプといったフィールドの タイプ を元に作られます。 Symfony2 には、アプリケーションで使える数多くのフィールドタイプが標準で組み込まれています。

サポートされているフィールドタイプ¶

次のフィールドタイプは、Symfony2 でネイティブにサポートされています:

テキストフィールド¶

  • text
  • textarea
  • email
  • integer
  • money
  • number
  • password
  • percent
  • search
  • url

選択フィールド¶

  • choice
  • entity
  • country
  • language
  • locale
  • timezone
  • currency

日付フィールド、時刻フィールド¶

  • date
  • datetime
  • time
  • birthday

その他のフィールド¶

  • checkbox
  • file
  • radio

フィールドグループ¶

  • collection
  • repeated

Hidden フィールド¶

  • hidden

ボタンフィールド¶

  • button
  • reset
  • submit

Base フィールド¶

  • form
