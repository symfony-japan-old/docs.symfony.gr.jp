email フィールドタイプ
======================

``email`` フォールドはHTML5の ``<input type="email" />`` タグで出力されるテキストフィールドです。

+-------------+---------------------------------------------------------------------+
| 対応するタグ| ``input`` ``email`` フィールド(テキストボックス)                    |
+-------------+---------------------------------------------------------------------+
| 継承された  | - `max_length`_                                                     |
| オプション  | - `required`_                                                       |
|             | - `label`_                                                          |
|             | - `trim`_                                                           |
|             | - `read_only`_                                                      |
|             | - `disabled`_                                                       |
|             | - `error_bubbling`_                                                 |
|             | - `error_mapping`_                                                  |
|             | - `mapped`_                                                         |
+-------------+---------------------------------------------------------------------+
| 親タイプ    | :doc:`form </reference/forms/types/form>`                           |
+-------------+---------------------------------------------------------------------+
| クラス      | :class:`Symfony\\Component\\Form\\Extension\\Core\\Type\\EmailType` |
+-------------+---------------------------------------------------------------------+
